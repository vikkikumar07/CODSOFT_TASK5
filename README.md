# Student Course Registration System

## Description
A simple Java console-based application that allows students to register for courses, view available courses, remove registered courses, and manage course enrollment.

## Features
- View available courses
- Register for a course
- Remove a registered course
- View registered courses
- Course capacity management
- Menu-driven interface

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList
- Scanner Class

## Project Structure
- Course.java
- Student.java
- CourseRegistrationSystem.java

## How to Run
1. Compile all Java files.
2. Run `CourseRegistrationSystem.java`.
3. Select options from the menu.

## Sample Menu
1. View Courses
2. Register Course
3. Remove Course
4. View Registered Courses
5. Exit
